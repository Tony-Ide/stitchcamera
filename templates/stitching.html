üîó Python API Access

Use these endpoints to access stitched frames from your Python code:

GET /api/frame/stitched
Get a single stitched frame as base64-encoded JPEG
GET /api/stitched/stream
Get continuous MJPEG stream of stitched cameras
GET /api/stitch/status
Get current stitching status and camera availability

üêç Python Example Code:

# Example: Get stitched frames from Python import requests import cv2 import numpy as np import base64 # Get a single stitched frame response = requests.get('http://localhost:5000/api/frame/stitched') if response.status_code == 200: data = response.json() frame_data = data['frame'].split(',')[1] frame_bytes = base64.b64decode(frame_data) frame = cv2.imdecode(np.frombuffer(frame_bytes, np.uint8), cv2.IMREAD_COLOR) # Process the stitched frame cv2.imshow('Stitched Frame', frame) cv2.waitKey(1) # Get stitching status status = requests.get('http://localhost:5000/api/stitch/status').json() print(f"Stitching enabled: {status['enabled']}") print(f"Homography locked: {status['locked']}")
